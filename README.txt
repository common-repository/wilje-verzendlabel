Connect your Woocommerce shop with the PostNL API using this simple solution provided by Wilje Online!

== Description ==

Effortlessly integrate your WooCommerce shop with the PostNL API using this plugin.
With just a few clicks, you can generate shipping labels for all types of packages, including letterbox parcels, ensuring a seamless shipping process.
In addition to label creation, the plugin allows you to send automated Track & Trace emails to your customers, keeping them informed every step of the way.
Simplify your order fulfillment and never worry about shipping hassles again

== Installation ==
Here's how you can easily install our plugin:

1. Install using Wordpress' own download directory
2. Activate the plugin through the 'Plugins' menu in WordPress - search for "Wilje Online PostNL".
3. Navigate to Woocommerce -> Settings -> Shipping -> Wilje Online PostNL
4. Fill in all fields. You will need an active PostNL account for using this plugin, which you can make [here](https://www.postnl.nl/en/business-solutions/ "PostNL"):

== Frequently Asked Questions ==

= What can this plugin do? =

This plugin enables you to directly print PostNL Shipping labels from the Order page.
With it you can print Labels for all 4 package types, including letterbox parcels.

= Does this plugin send Track & Trace codes? =

Yes it does! Upon Order completion the plugin sends a small email detailing that a Track & Trace code has been sent.
This email can be send manually as well on the Order page.
